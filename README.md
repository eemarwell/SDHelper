Java SDHelper
=============
![Java CI with Maven](https://github.com/eemarwell/SDHelper/workflows/Java%20CI%20with%20Maven/badge.svg?branch=master)

O Java SDHelper foi desenvolvido para auxiliar e melhorar o fluxo de trabalho dos agentes do Service Desk Unisys, dentro do Projeto Heineken. Projeto de aplicativo privado inicialmente para desenvolvimento do sistema específico para o projeto Heineken, após ser desenvolvido e criado de forma específica, será distribuído em código aberto.

>O SDHelper foi desenvolvido utilizando JavaFX, utilizando fluxo de Scenes com arquivos de visualização em FXML.


#### O SDHelper foi desenvolvido utilizando as seguintes tecnologias:

1. **Maven** - _Projeto utiliza o Maven para gerenciamneto de dependências._
2. **JavaFX** - _Além de ser utilizado o Java 1.8, a tecnologia JavaFX foi aplicada para que o projeto tivesse um cara melhor e que as views fossem feitas de forma facilitada._
3. **Intellij IDEA** - _O IDEA foi utilizado para melhor produtividade, a pasta de configuração do IDEA está presente no projeto git para caso seja utilizado, já sejam aplicadas as configurações necessárias._
4. **Hibernate** - _O Hibernate foi utilizado para facilitar a conexão com o banco de dados._
5. **MySQL** - _O banco de dados escolhido foi o MySQL versão 8.0._
6. **Spring** - _Os seguintes componentes do Spring foram utilizados para configuração rápida do projeto e abstração de segurança, rápida criação de projeto._
* **Spring Lombok** - _Para a configuração facilitada as entidades._
* **Spring Boot** - _Para a rápida configuração e inicialização do projeto._
* **Spring Batch** - _Para a criação rápida e prática de tarefas agendadas no projeto._
